
  # Emergency Response Dashboard

  This is a code bundle for Emergency Response Dashboard. The original project is available at https://www.figma.com/design/HsiasVk84jggze3vrULhK0/Emergency-Response-Dashboard.

  ## Running the code

  Create `src/app/config/ai.ts` from `src/app/config/ai.example.ts` and set your OpenRouter API key.

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
